# Pendulum Plotter

Creative art project simulating a paint bucket swinging on a spherical pendulum in 3D space, dripping paint onto a 2D canvas below.

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Run linting
pnpm lint

# Format code
pnpm format
```
